Front-end web development is the practice of converting data to a graphical interface, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that data. --Wikipedia

In order to do front end development, one should learn the following technologies:

  • Git
  • Bash
  • Markdown
  • HTML5
  • Pug, or another HTML preprocessor
  • CSS3
  • SASS/SCSS, or another CSS preprocessor
  • JavaScript
  • TypeScript
  • React/JSX or Angular, maybe Vue
  • Redux
  • Mocha
  • Chai
